קבוצות האותיות המקסימליות בהן כל זוג אותיות בכל סדר אפשרי מהווה מילה:

דמנצ"ר, דמנר"ש, דמצק"ר, דמקר"ש, חכנפ"ר, חמנצ"ר, חמנר"ש, חמצק"ר, חמקר"ש, חנפצ"ר, חנפר"ש

Maximal letter sets where every pair of letters in every possible order forms a word.

קבוצות האותיות המקסימליות בהן כל זוג וכל שלישיית אותיות בכל סדר אפשרי מהווה מילה:

בה"ל, בע"ר, גמ"ש, דל"ק, חל"ש, חמ"ק, חצ"ר, מצ"ר, מר"ש, ער"ש, פר"ש

Maximal letter sets where every pair and every triplet of letters in every possible order forms a word.

מהן, אלו שלא נדרשות לשימוש בתחיליות מש"ה וכל"ב:

חצ"ר, מצ"ר, פר"ש

Those of the former that do not require the use of prefixes.

קבוצת האותיות המקסימלית בה כל שלישיית אותיות בכל סדר אפשר מהווה מילה:

וחל"ש

The maximal letter set where every triplet of letters in every possible order forms a word.

קבוצת האותיות המקסימלית בה כל רביעיית אותיות בכל סדר אפשר מהווה מילה:

וכמ"ס

The maximal letter set where every quartet of letters in every possible order forms a word.

לא מצאתי קבוצת אותיות בה כל חמישייה או יותר של אותיות בכל סדר אפשרי מהווה מילה.

I could not find a letter set where every quintet or more of letters in every possible order forms a word.
